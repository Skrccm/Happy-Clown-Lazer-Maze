Shortcuts
-All buttons in the main menu can be accessed with ALT 1-5
-All buttons in the pause menu can be accessed with ALT 1-4
-The help file can be accessed with F1
Possible Errors
-Will continue to try to generate a new highscores, but none can be created.
 -Probably does not have write rights. Make sure it is in a folder with write access.
-Laser or highscore menu will become extremely slow.
 -Not sure why. Probably memory error.
